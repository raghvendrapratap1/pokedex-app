Pokédex — React TDD Application

A modern Pokédex application built using React, focused on clean architecture and strict Test-Driven Development (TDD). Every feature is developed through small, meaningful commits following the Red → Green → Refactor cycle, ensuring reliability, maintainability, and high code quality.

The app provides an interactive Pokémon browsing experience, including listing, search, filtering, and detailed views powered by PokéAPI. It also incorporates performance-focused features like infinite scrolling and a smooth UI with theme support.
